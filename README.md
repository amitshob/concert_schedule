Concert Schedule App
By Amit Mhatre

**Description**

This App allows you to track bands and the venues where they've played concerts.


**Setup**-

* Clone repository
* Install Postgres and have it working



***Technologies Used**
Ruby,rspec,Sinatra,Capybara,Postgres,Activerecord, Rake
